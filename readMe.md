# Các công cụ sử dụng
## Phần mềm
1. Keil uVision5 v5.29.0.0
2. STM32CubeMX Version 6.2.1
4. Visual Studio 2019
## Phần cứng
1. Kit STM32F103C8T6
2. Mạch nạp STLINK V2
3. Module điều khiển động cơ L298N
4. LCD 16x2
5. Module chuyển đổi I2C sang LCD
6. Motor Encoder 334
7. Cáp USB TO COM PL2303 V2
8. Biến trở 10k
